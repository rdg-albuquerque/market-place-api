# rest-api-validacoes-yup

## Características da API

* CRUD de usuários e produtos relacionados ao usuário com banco de dados relacional
* Cadastro de usuários com criptografia de senha
* Login de autenticação e geração de token
* CRUD de produtos e atualizações do perfil do usuário protegidas por autênticação(Token requerido).
* Validações do body da requisição com a biblioteca YUP.
